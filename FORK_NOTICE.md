# Zephyr Fork Notice

Zephyr is an unofficial, community-maintained fork of
[Helium](https://github.com/imputnet/helium), created and maintained by
QuanticStudios.

## Scope

- Build on top of Helium's core browser work.
- Prioritize UI and UX improvements.
- Keep compatibility with existing Helium protocol usage (including
  `helium://setup`) unless explicitly documented otherwise.

## Attribution and upstream

- Helium remains the upstream project this fork is based on.
- This fork preserves upstream credits and licensing notices.
- Contributions suitable for upstream are intended to be proposed back to Helium.

  clarity.
